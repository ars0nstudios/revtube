FluentBS, from clipIt.

FluentBS is a Windows 11 inspired fork of Bootstrap 1.
It includes base CSS and HTML for typography, forms, buttons, tables, grids, navigation, and more.

Fun fact: FluentBS started out as a Stylus theme.

About FluentBS Brief history, browser support, and more

History

clipIt was due for a more modern looking UI. Well, the developer is pretty good with CSS, and Windows 11's UI looks quite nice. So he thought, why not turn it into a CSS theme?

Follow @InnsbruckTaos

Browser support

Bootstrap is tested and supported in major modern browsers like Chrome, Safari, Internet Explorer, and Firefox.

  • Latest Safari
  • Latest Google Chrome
  • Firefox 4+
  • Internet Explorer 7+
  • Opera 11

What's included

FluentBS uses the same Javascript plugins as Bootstrap v1.4.0.

  • Javascript plugins
  • All original .less files
  • Fully compiled and minified CSS
  • Complete styleguide documentation
  • Three example pages with different layouts
